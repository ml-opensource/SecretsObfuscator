# Secrets Obfuscator

A simple script that obfuscates secrets.

## How to use this with SPM?

First, add SecretsObfuscator to the dependencies.

```swift
    dependencies: [
        .package(url: "https://github.com/nodes-ios/SecretsObfuscator", branch: "develop")
    ],
```

Then create a json file with the secrets, here's an example:

```json
{
  "mixpanel": "99jf4jf3nd61lcm6ik1nc8c91l3m5ndkahci23nddl4",
  "firebase": "47dh1kdl39dh1995465lkj31ldaghj2312mlk12l4k3",
  "nstack": "jfk39d8djka09093kalcbxndk1ldeko12mxamnx12ajd8",
  "crashes": "kfkk398djakjdnzxvb12111jhdeundi89smmxnkjh31k"
}
```

then excecute the following command:

```bash
swift package obfuscate-secrets -k <SomePassword> <Secrets json file> <Outputfile>
```

Example:

```bash
$ swift package obfuscate-secrets -k WFidKGBv512H secrets.json SData.swift
```
